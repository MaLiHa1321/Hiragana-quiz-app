# 🎌 Hiragana Quiz App

A simple, interactive web app to learn and practice Japanese Hiragana characters through timed quizzes.

## Live Demo

[View Live Demo →](https://hiragana-quiz-app-seven.vercel.app)

## Features

- 🎯 Randomized quiz questions from 46 Hiragana characters
- ⏱️ Adjustable time limits with Easy, Medium, Hard difficulty
- 📊 Track your score, accuracy, and answer streak
- 📈 Mini performance chart showing recent answers
- 🌙 Dark mode toggle
- 📱 Fully responsive design for mobile and desktop
- ⌨️ Keyboard support (press Enter to check answer)
- 🎉 Confetti effects on correct answers

## How to Play

1. Choose your settings (question count, time limit, difficulty)
2. Click "Start Quiz"
3. Type the romaji reading of the displayed Hiragana
4. Click "Check" or press Enter
5. Click "Next" to continue
6. Complete all questions to see your final score

## Technologies Used

- HTML5
- CSS3 (Glass morphism, animations)
- JavaScript ES6
- Canvas API for charts
- Canvas Confetti library
